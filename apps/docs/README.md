# FileBase Docs

User-facing documentation for FileBase. Until a docs site framework is selected, guides live as markdown under [`content/guides`](./content/guides) and can be read directly on GitHub.

## Guides

1. [Quick Start](./content/guides/01-quick-start.md) — one-command install on a Linux server.
2. [Manual Docker Compose](./content/guides/02-docker-compose.md) — clone, configure, and run the stack yourself.
3. [First-Run Onboarding](./content/guides/03-first-run-onboarding.md) — guided in-browser setup after install.
4. [Local Storage](./content/guides/04-local-storage.md) — writing files to the FileBase host filesystem.
5. [FTP Storage](./content/guides/05-ftp-storage.md) — uploading to a remote FTP server.
6. [SFTP Storage](./content/guides/06-sftp-storage.md) — uploading over SSH with passwords or keys.
7. [API Keys](./content/guides/07-api-keys.md) — server-to-server authentication.
8. [Upload Presets](./content/guides/08-upload-presets.md) — reusable validation and transformation rules.
9. [Signed Uploads](./content/guides/09-signed-uploads.md) — secure browser and mobile uploads.
10. [SDKs](./content/guides/10-sdks.md) — JavaScript/TypeScript packages for client and server.
11. [Troubleshooting](./content/guides/11-troubleshooting.md) — common failure modes and how to diagnose them.
12. [Security Recommendations](./content/guides/12-security.md) — hardening checklist for production deployments.
